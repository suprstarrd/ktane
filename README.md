# Keep Talking and Nobody Explodes Manual

I play [Keep Talking and Nobody Explodes](https://www.keeptalkinggame.com) with some friends, and some of them have had a bit of trouble with the manual. I also want to make it easier for us to get into mods later down the road once we're better. With that in mind, I've been working on making this manual. I hope to improve it for general use, and I provide my work as a reference and baseline for anyone else who wants to get started with independent manual development.
